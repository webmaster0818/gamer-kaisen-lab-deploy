📊 ジッターとは？ping値との違い

ジッターとは何か？

ジッター（Jitter）とは、ping値の揺れ幅のことです。 ping値が「平均的な遅延」を表すのに対し、ジッターは「遅延のばらつき」を表します。

ジッターが低い場合（2ms）

ping: 10ms → 12ms → 11ms → 10ms
通信が安定。ゲームが滑らか。

ジッターが高い場合（15ms）

ping: 10ms → 25ms → 8ms → 30ms
通信が不安定。カクつき・ワープ発生。

ping値とジッターの違い

項目ping値ジッター値
意味平均遅延時間遅延の揺れ幅
影響操作の反応速度通信の安定性
FPS推奨値15ms以下3ms以下
症状（高い場合）操作が遅れるカクつき・ワープ
